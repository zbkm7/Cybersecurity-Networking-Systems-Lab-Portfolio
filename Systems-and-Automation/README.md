## Tools & Technologies
- Bash
- Linux command-line utilities
- Python
- GDB
- Virtualized environments

## Project Structure
Each subfolder includes:

- Supporting scripts or outputs
- Documentation demonstrating problem-solving and system understanding

These projects emphasize practical system administration and automation skills commonly used in IT operations and cloud environments.

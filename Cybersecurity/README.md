# Cybersecurity Projects & Labs

This folder contains hands-on cybersecurity projects, labs, and technical reports completed as part of undergraduate coursework and independent study.

## Topics Covered
- Web application security (DVWA, SQL injection, authentication flaws)
- Vulnerability analysis and exploitation
- Secure coding principles
- Password hashing and cracking techniques
- Incident response and breach analysis
- Malware and ransomware awareness
- Identity and access control concepts

Each subfolder includes documentation, screenshots, and reports demonstrating practical security skills and analytical understanding.

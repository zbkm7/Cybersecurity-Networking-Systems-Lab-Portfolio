# DVWA Web Application Security Lab

This project documents hands-on testing and exploitation of common web application vulnerabilities using Damn Vulnerable Web Application (DVWA).

## Overview
DVWA is an intentionally vulnerable web application designed to help security practitioners understand how common web attacks work in practice.

## Vulnerabilities Explored
- SQL Injection
- Authentication and session weaknesses
- Input validation flaws
- Improper error handling
- Privilege escalation concepts

## Environment
- DVWA hosted locally
- Web browser testing
- Manual exploitation techniques
- Supporting screenshots and analysis included in the report

## Artifacts
- DVWA Security Report (PDF)
- Screenshots documenting vulnerabilities and results

This lab demonstrates practical web application security testing and foundational penetration-testing concepts.
